#  Code Speed – CSE299 Project

A **Customizable and Dynamic Typing Test for Programmers** designed to improve coding speed and accuracy through structured practice and real-time feedback.

---

##  Project Overview

**Code Speed** is a web-based application that helps programmers enhance their typing skills specifically for coding environments. The system provides typing tests, tracks performance, and adapts to user progress.

---

##  Features

*  Adaptive typing tests for programmers
*  Real-time speed (WPM) and accuracy tracking
*  Performance results and history
*  User authentication (Register/Login)
*  Data stored using database
*  Multiple programming language support

---

##  How to Run the Project

###  Clone the Repository

```bash
git clone https://github.com/your-username/code-speed.git
cd code-speed
```

---

###  Setup Backend

```bash
cd backend
npm install
npm start
```

Server will run on:

```
http://localhost:5000
```

---

###  Setup Frontend

```bash
cd frontend
npm install
npm start
```

Frontend will run on:

```
http://localhost:3000
```

---

##  Testing the Application
